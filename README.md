# 🍯 Del Fogón — Landing Page

Proyecto para el taller de Git y Github. Un landing page profesional y responsivo para un negocio de productos caseros artesanales.

## 📋 Descripción

**Del Fogón** es un sitio web de presentación para un negocio que vende productos caseros de calidad. La página está diseñada con un enfoque moderno, elegante y totalmente responsivo, optimizada para convertir visitantes en clientes.

## ✨ Características

- **Navegación responsive**: Barra fija con menú hamburguesa para dispositivos móviles
- **Sección Hero**: Titular atractivo con call-to-action y estadísticas
- **Catálogo de productos**: Galería de productos con descripciones
- **Sección "Por qué elegirnos"**: Destacados de la marca
- **Historia**: Narrativa de los orígenes del negocio
- **Testimonios**: Opiniones de clientes satisfechos
- **Formulario de contacto**: Integración con WhatsApp para pedidos
- **Animaciones scroll reveal**: Efectos visuales al hacer scroll
- **Efecto glass-morphism**: Diseño moderno en la navegación
- **Paleta de colores cálida**: Tonos terracota, oliva y miel

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Diseño responsivo y animaciones
- **JavaScript Vanilla**: Interactividad (sin dependencias)
- **Google Fonts**: Tipografías elegantes (Playfair Display y DM Sans)

## 💡 Funcionalidades JavaScript

- **Efecto de scroll**: Cambio de estilos en la nav al hacer scroll
- **Menú responsivo**: Hamburguesa que abre/cierra menú en móvil
- **Scroll Reveal**: Animaciones de entrada de elementos con Intersection Observer
- **Validación de formulario**: Envío de pedidos vía WhatsApp

## 📱 Responsive Design

La página se adapta perfectamente a:
- 📱 Móviles (320px+)
- 📱 Tablets (768px+)
- 🖥️ Escritorio (1024px+)

## 🚀 Cómo Usar

1. Clonar el repositorio
2. Abrir `index.html` en el navegador
3. El sitio es completamente funcional sin necesidad de servidor
